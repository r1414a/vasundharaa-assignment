### Data Table
* Displayed paginated project data from a mock API
* Columns:- Project Name, Latitude, Longitude, Status, Last Updated
* Built using MUI DataGrid, So sorting is in-built
* Pagination (10 / 20 / 50 rows per page)
* Row highlighting on selection

### Map Integration
* Built using React Leaflet
* Plots markers based on latitude & longitude
* Clicking a table row highlights & zooms to the marker
* Clicking a marker highlights & scrolls to the table row

### Filtering & Searching
* Status filter select input for (All / Active / In Progress / Completed)
* Category filter select input for (Energy, Infrastructure, Technology, etc.)
* Search by project name, status or category, filtered result shown in table & map.

### User Registration Form
* Proper validation on Email, user ID, password fields.
* Proper mentioning of errors in case of wrong input.
* Show / Hide password toggle for passwords

### Tech Stack
* React.js
* MUI DataGrid
* React Leaflet
* JavaScript (ES6+)
* CSS / Tailwind CSS
* Mock API (JSON)
* React icons
